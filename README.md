# 宝宝性别竞猜 - 多人在线版

🎉 一个支持多人实时在线投注的宝宝性别竞猜游戏！

## 🌐 在线访问

部署完成后，你的网址将是：`https://你的GitHub用户名.github.io/baby_bet/`

## ✨ 功能特点

- ✅ **实时同步** - 所有人的投注立即可见，无需刷新
- ✅ **多人在线** - 支持无限人数同时投注
- ✅ **数据永久保存** - 使用 Firebase 云端存储
- ✅ **移动端适配** - 手机、平板完美显示
- ✅ **开奖功能** - 自动计算中奖人数和金额

## 🚀 技术栈

- 纯 HTML + CSS + JavaScript（无需编译）
- Firebase Realtime Database（实时数据库）
- GitHub Pages（静态网站托管）

## 📱 使用说明

1. 访问网页
2. 输入你的名字
3. 选择男孩或女孩
4. 输入投注金额
5. 提交投注
6. 等待开奖！

## 🔧 本地开发

如果你想在本地测试：

```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js
npx http-server
```

然后访问 `http://localhost:8000`
